## About The Project

This repository serves as a personal log and project space for my journey into the world of machine learning with Scikit-Learn. The primary source of learning is the fantastic content provided by the Ryan and Matt Data Science YouTube channel. Here, I'll be documenting my progress, experimenting with different algorithms, and applying concepts to various datasets.

This is more than just a code repository; it's a chronicle of my learning process, a place to practice, and a portfolio of my growing skills in data science.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This project uses `uv` as a package manager. You can install it by following the official instructions.

### Setup

1. Clone the repo
   ```sh
   git clone https://github.com/drod75/scikit-learn-learning.git
   ```
2. Create and sync the virtual environment
   ```sh
   uv venv && uv sync
   ```

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## Usage

The primary use of this repository is to follow along with the tutorials and examples. The `notebooks` directory contains Jupyter notebooks for each topic.

To start the Jupyter Lab server, run the following command:
```sh
uv run jupyter lab
```
